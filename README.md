# Electron-Proton Cross Section Calculator
This script is designed to facilitate calculations on the electron-proton
differential cross section in the lab-frame, and, in the future, the center-of-mass frame
## Setup
### Clone repository:
- `git clone https://github.com/rlatosky/ep-cscalc.git`
### Setup virtual environment:
- `python -m venv ./venv`
- `source ./venv/bin/activate`
### Install required pip packages:
- `pip install numpy argparse`
### Run script:
- `python ep-cscalc.py -h`
